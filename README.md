# Eve-ng-lab
Eve ng  lab running : 
 -network configurations, users authentication and vpn access 
 - This lab demonstrates a basic OSPF routing setup.
 - Different Sophos security features (Acl Rules , Vlans)

### Requirements
- EVE-NG Community Edition
- Required VM Images:
  IOS : 
  - i86bi_linux_l2-adventerprisek9-ms.SSA.high_iron_20180510.bin
  - L2-ADVENTERPRISEK9-M-15.2-20150703.bin
  - `vPC` (Virtual PC)
  - Images of diffrenet system (kali , MX linux , Rocky Linux , ubuntu server , windows server )
  - Sophos VI-19.5.0_GA.KVM-197
### How to Use
1. Import the `Lab-Name.zip` file using the EVE-NG Import function.
2. Assign the appropriate QEMU images to the nodes.
3. Start the devices!
